# glml
GLML - WebGL Markup Language (proof-of-concept)

## Example

```
<gl-ml ver='100' width="1024" height="768" clearColor="#777777" camera="35">
  <gl-box x=1 y=1 z=1 material="#00ff00" />
  <gl-light type='spot' shadow color="#FF7F00" x=-50 y=20 z=70 />
</gl-ml>
```
